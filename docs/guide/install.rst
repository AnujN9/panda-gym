.. _install:

Installation
============

With pip
--------

To install ``panda-gym`` with pip, run:

.. code-block:: bash

    pip install panda-gym

From source
-----------

To install ``panda-gym`` from source, run:

.. code-block:: bash

    git clone https://github.com/qgallouedec/panda-gym/
    cd panda-gym
    pip install -e .
